**** EMULATION GUIDE ****

Getting Started

This guide explains how to emulate the Commodore 64 and Amiga using recommended emulators.

Commodore 64 (VICE)

Download VICE from the official build archives.
Load a `.d64`, `.t64`, or `.prg` file to begin.

Amiga (WinUAE / FS-UAE)

Use Kickstart ROMs and ADF disk images.
Configure CPU, Chip RAM, and audio for best results.

Troubleshooting

If games run too fast, enable “Exact Cycle Emulation”.
If audio crackles, reduce latency settings.
